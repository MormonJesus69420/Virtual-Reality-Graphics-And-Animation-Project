Detailed Description

template<typename T, int n, int m>
class GMlib::Simplex< T, n, m >

interface for the Simplex, LineSegment, Triangle, Tetrahedron classes.

A template (n x m) Simplex, the simplex is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions

The template n denote the dimention of the space the simplex is embedded into. The temlate m denote the simplex dimention, i.e. m=2 is a line segment, m=3 is a triangle etc.
