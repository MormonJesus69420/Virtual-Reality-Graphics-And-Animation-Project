Detailed Description

template<typename T, int n>
class GMlib::LineSegment< T, n >

The Static LineSegment class.

A template LineSegment, the LineSegment is static i.e. the dimentions of the embedding space can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions
