Maps a m-dimensional index to nD.

void sqMapping( const T& index_1d, T* index ) Maps the m-variat index into a n-variat index.

The total number of elements in both the m- and n-dimensional space must match, ( m * res_fact_m == n * res_fact_n ).

Usage example:

Template Parameters
TIndexing typename, e.g.: short, int, unsigned int.
mIndex dimension.
res_fact_mResolution factor. Described as 2^(res_fact_m - 1).
nIndex dimension.
res_fact_nResolution factor. Described as 2^(res_fact_n - 1).
Parameters
[in]index_mPointer to the m-dimensional index. Array of m T-values.
[out]index_nPointer to the n-dimensional index. Array of n T-values. Must be initialized.
[out]errHolds an error status.
