A method for computing N dimensional recursive divided differenses on a parametric-regularized dataset. The container must be a 2*N dimenional container where the first N dimensions represent the dimensions of the dataset, whilest the second set of N dimensions represent the derivatives. Memory must be pre-allocated. The G& operator [] ( int idx ) must exist as an access operator for each dimension. The data itself must have the H& operator * (double d). Positional data must exist as a minimum for each data point.

Parameters
[in,out]pThe data set
[in]sizesSize of each of the N basic dimensions of p
[in]dtThe delta of two adjacent datapoints in a given dimension
[in]closedWhether the data set is ... closed in a given dimension
[in]dThe number of derivatives to compute in a given dimension
[in]edNumber of existing derivitives in a given dimension
