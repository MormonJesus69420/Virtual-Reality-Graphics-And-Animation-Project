Detailed Description

template<typename T, int n>
class GMlib::Box< T, n >

The Static Box class A template Box, the Box is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions.

The Box is described by a minimum Point and a maximum Point (the diaginal).
