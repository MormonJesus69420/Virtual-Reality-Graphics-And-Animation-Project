virtual void PSurf<T,3>::eval( T u, T v, int d1, int d2, bool lu , bool lv ) const = 0 Surface evaluator, the formula of the surface definition Requires implementation in PSurf sub-classes.

Parameters
[in]uEvaluation parameter in u-direction.
[in]vEvaluation parameter in v-direction.
[in]d1Number of derivatives to be computed for u.
[in]d2Number of derivatives to be computed for v.
[in]lu(default true) Whether to evaluate from left (or right) at u.
[in]lv(default true) Whether to evaluate from left (or right) at v.
