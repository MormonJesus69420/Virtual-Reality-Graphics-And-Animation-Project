Detailed Description

template<typename T, int n>
class GMlib::Vector< T, n >

The Static Vector Class.

A template Vector, the Vector is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions.

The vector is only using n values, but it is acting as it is in homogenious coordinates. The (n+1).nt element, which is not there, is actually implicite 0.
