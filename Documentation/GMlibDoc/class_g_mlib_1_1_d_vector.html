Detailed Description

template<typename T>
class GMlib::DVector< T >

Interface for the Dynamic Vector class.

Interface for the Dynamic Vector class. DVector (Dynamic Vector) is a container class which can dynamic change dimention. together with DMatrix a lot of standard vector/matrix operations are available.

Increasing the dimention of the vector.
