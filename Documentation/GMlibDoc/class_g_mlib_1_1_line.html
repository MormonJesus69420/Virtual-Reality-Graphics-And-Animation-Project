Detailed Description

template<class T, int n>
class GMlib::Line< T, n >

The Static Line class.

The Static Line class. A template Line, the Line is static i.e. the dimentions of the embedding space can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions.
