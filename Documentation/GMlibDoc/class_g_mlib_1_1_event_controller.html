Detailed Description

EventController base class.

Provides an interface to detect and control Events that can happen to SceneObjects during a time step dt.

Inherited classes need to implement functions which detects

  1. Events within a given dt (first pass)
  2. Any events following after handling of a specific event within that dt (second pass) NB! When storing events in Array, use insertAlways for performance reasons.

Optionally, inherited classes can store customized information and perform updates based on events.
