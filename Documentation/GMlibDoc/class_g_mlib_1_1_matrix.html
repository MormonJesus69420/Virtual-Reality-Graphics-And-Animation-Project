Detailed Description

template<typename T, int n, int m>
class GMlib::Matrix< T, n, m >

The Static Matrix class.

A template (n x m) matrix (n rows and m columns), the matrix is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions
