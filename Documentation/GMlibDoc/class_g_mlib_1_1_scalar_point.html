Detailed Description

template<typename T, int n>
class GMlib::ScalarPoint< T, n >

The Static ScalarPoint class A template ScalarPoint, the ScalarPoint is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions.

The ScalarPoint is a Point and a conected scalar value.
