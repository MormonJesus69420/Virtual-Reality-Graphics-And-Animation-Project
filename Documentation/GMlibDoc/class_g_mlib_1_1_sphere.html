Detailed Description

template<typename T, int n>
class GMlib::Sphere< T, n >

The Static Sphere class A template Sphere, the Sphere is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions.

The Sphere is a centre Point and a conected scalar value, radius. It also inclue a boolean telling if the Sphere is valid or not.
