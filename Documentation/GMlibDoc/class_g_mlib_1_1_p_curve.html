bool PCurve<T,n>::getClosestPoint(const Point<T,n>& q, T& t, Point<T,n>& p, double eps, int max_iterations) const To find closest point on a curve from a point Using a 2nd degree taylor expansion <c-p + dt*c' + 0.5dt^2*c'' , c' + dt*c''> = 0

Parameters
[in]qThe point to be close to
[in]tThe parameter value at closest point, bout guess and result (also return value)
[out]pThe closest point on the curve
[in]eps(default 10e-6) The tolerance (in parametric space)
[in]max_iterations(default 20) The maximum number of iterations in the algorithm
Returns
"true" if a closest point is found "false" if not.
