Detailed Description

template<typename T, int n>
class GMlib::SqMatrix< T, n >

The Static Square Matrix class.

class SqMatrix: public Matrix<T,n,n> A template (n x n) matrix, the matrix is static i.e. the dimentions can not be change. The template type must be clean, i.e. is not allocating memory and without any virtual functions. This Square Matrix inherit from the static Matrix class
