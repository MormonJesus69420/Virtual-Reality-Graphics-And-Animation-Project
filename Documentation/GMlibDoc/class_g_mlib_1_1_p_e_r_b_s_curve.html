void PERBSCurve<T>::setPartitionCriterion(int pct) To set a criterion for how to split the curve into several partitions This can ensure discontinuities at different levels are displayed

Parameters
pctpartition criterie (valid values are 0 and 1. pct = 0 - split where we have a discontinue function (multiplisity = order = 2) pct = 1 - split in every single knot
